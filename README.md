# pdf2md

A simple and lightweight CLI tool that converts PDF files into clean Markdown using MarkItDown.

---

# ⚡ Features

- PDF → Markdown conversion
- Clean CLI interface
- Save output to file or folder
- Auto-generated filenames
- Lightweight and fast
