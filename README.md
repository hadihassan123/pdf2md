# pdf2md

A simple CLI tool that converts PDF files into clean Markdown using MarkItDown.

## Features
- PDF → Markdown conversion
- CLI interface
- Output to file or folder
- Auto filename generation

## Usage

```bash
pdf2md file.pdf
pdf2md file.pdf -o output.md
pdf2md file.pdf -o ./out/
